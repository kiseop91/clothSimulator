# Hockey Drill Studio — 팀/커뮤니티/채팅 기능 추가

## Context

현재 Hockey Drill Studio는 개인 사용자 중심 SaaS (Supabase + Express + React)로, 유저별 드릴/세션 관리, Free/Pro 구독, AI 생성 기능이 구현되어 있다. 아마추어 하키 팀 단위 사용, 팀 간 교류, 드릴 공유 커뮤니티를 위해 4가지 핵심 기능을 추가한다:

1. **팀 플랜** — 팀 생성/관리, 초대, 세션 공유, 알림
2. **드릴 공유** — 커뮤니티 게시판, 좋아요, 가져오기
3. **팀 매칭** — 연습경기 상대 모집 게시판
4. **DM 채팅** — 유저 간 1:1 메시지 + 팀 그룹채팅

## 비즈니스 모델 & 핵심 전략

### 타겟: 플레이어가 주 고객, 코치는 무료로 끌어들임

이 서비스는 **코치에게 파는 도구가 아니다.** 하키 연습에서 "오늘 드릴 뭐야?"를 모르고 가는 **플레이어**가 주 타겟이다. 플레이어가 드릴 정보를 보고 싶어하고, 연습 전에 예습하고 싶어한다.

### 코치 확보 전략: "5명이 모이면 감독님을 초대할 수 있어요!"

```
플레이어 5명 가입 → 팀 결성 → 코치 초대 기능 해금
→ 코치는 완전 무료 (드릴 생성, AI 생성, 팀 관리 전부)
→ 코치가 만든 드릴/세션을 팀원이 열람
→ 팀원이 가치를 느끼고 유료 전환
```

**코치를 모으는 게 핵심.** 코치가 들어오면 콘텐츠(드릴)가 생기고, 플레이어에게 가치가 생긴다.
유저도 코치 역할을 할 수 있다 (역할 전환 가능).

### 수익 구조

| 대상 | 플랜 | 가격 | 설명 |
|------|------|------|------|
| **플레이어** | Free | ₩0 | 드릴 3개, AI 5회/일, 팀 가입 가능 |
| **플레이어** | Pro | ₩9,900/월 | 무제한 드릴, AI, 클라우드 동기화 |
| **팀** | Team | ₩49,900/월 | 팀원 전체 Pro 기능, 팀 관리 |
| **코치** | 무료 | ₩0 | 드릴 생성/AI/팀 관리 전부 무료 |
| **코치** | 후원 | 팁 | 플레이어/팀이 코치에게 후원 (플랫폼 수수료 10~20%) |

### 팀 결성 플로우

```
1. 플레이어가 가입 (Free)
2. "팀 만들기" → 최소 5명 플레이어 필요 (초대 링크 공유)
3. 5명 모이면 팀 결성 완료 → "코치 초대" 버튼 활성화
4. 코치 초대 (링크/코드) → 코치가 가입 (무료)
5. 코치가 드릴/세션 생성 → 팀원에게 자동 공유
6. 팀원이 모바일에서 "오늘의 드릴" 확인
```

### 코치 역할 특전 (무료)

- 무제한 드릴/세션 생성 + AI 생성
- 팀 관리 (로스터, 출석, 역할 관리)
- 커뮤니티 드릴 게시 (공유/평판 쌓기)
- 후원 수익 받기

### 후원 시스템

- 팀/개인이 코치 프로필에서 "후원하기" 클릭
- 1회성 후원 (₩5,000 / ₩10,000 / ₩30,000 / 자유 금액)
- 플랫폼 수수료 15% → 코치에게 85% 지급
- 후원 랭킹 / 인기 코치 표시 → 코치의 동기부여

## 핵심 설계 결정

1. **드릴 공유는 스냅샷 복사** — 원본 수정/삭제해도 공유본 유지
2. **채팅은 Supabase Realtime** — 별도 WebSocket 서버 불필요
3. **팀 플랜은 owner 기준** — 멤버는 개별 구독 불필요, owner의 tier='team'이 게이트
4. **매칭은 단순 게시판** — 자동 알고리즘 없이, 올리고 응답하는 방식
5. **BottomNav 공통 컴포넌트** — 10개+ 페이지로 확장되므로 모바일 내비 필수

### Playwright MCP 테스트 실행 방법

각 Phase 구현 완료 시:
1. Vite dev server (`npx vite`) + API server (`npx tsx server/index.ts`) 실행 중인지 확인
2. Playwright MCP로 `http://localhost:5173` 접속
3. 위 테스트 케이스를 순서대로 수행
4. 실패 시 → 수정 → 재테스트
5. 전부 통과 → 다음 Phase 진행
